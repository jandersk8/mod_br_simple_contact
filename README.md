## üá∫üá∏ English Description

**BR Simple Contact** is a lightweight contact form module designed specifically for the **Joomla 6** ecosystem.

It solves a persistent issue in CMS theming: CSS conflicts. By utilizing a unique **"Stand-alone CSS"** architecture, this module completely isolates its styles (Modern Card Design + Floating Labels) from your site's template. This ensures the form looks professional and functions perfectly on any Joomla 6 website, regardless of the active template or CSS framework (Bootstrap 5, Tailwind, etc.).

### üöÄ Key Features

* **Joomla 6 Native:** Optimized for the latest Joomla architecture.
* **Template-Proof:** Uses isolated IDs and styles. Your template cannot break the form layout.
* **Modern UI:** Clean "Card" container with "Floating Labels" (Material style) and native validation.
* **AJAX Submission:** Seamless email sending without page reloads.
* **Visual Feedback:** Built-in loading spinner and instant success/error alerts.
* **Zero Dependencies:** Does not rely on the template's Bootstrap version.

### üì¶ Installation

1.  Download the **`mod_br_simple_contact`** ZIP package.
2.  Log in to your Joomla 6 Administrator Panel.
3.  Go to **System** -> **Install** -> **Extensions**.
4.  Upload and Install the package.

### üõ†Ô∏è How to Use

1.  Navigate to **Content** -> **Site Modules**.
2.  Click **New** and select **BR Simple Contact**.
3.  Configure the basic settings (Recipient Email, etc.).
4.  Select a template position (e.g., `sidebar-right`, `footer`) and **Publish**.
5.  Assign the module to the desired pages under the "Menu Assignment" tab.
